2. What additional pages did you add?

3. What additional features did you add?

We have implemented the following additional features:

  • Groups: Users can create and join groups based on their interests.
  • Photo Gallery: Users can upload and share photos in albums.
  • Events: Users can create and RSVP to events or gatherings.
  • Badges/Achievements: Users can earn badges for specific activities on the platform.

6. Specify all the different (server) routes your app supports

Our app supports the following server routes:

  • GET /: Returns the homepage of the social network.
  • GET /about: Displays the "About Us" page.
  • GET /contact: Displays the "Contact" page.
  • GET /privacy: Displays the "Privacy Policy" page.
  • GET /terms: Displays the "Terms of Service" page.
  • GET /groups: Displays a list of groups and allows users to create or join groups.
  • POST /groups: Handles the creation of a new group.
  • GET /gallery: Displays user photo albums and allows photo uploads.
  • POST /gallery: Handles the upload of photos to user albums.
  • GET /events: Displays upcoming events and allows users to create or RSVP to events.
  • POST /events: Handles the creation of a new event.
  • POST /events/rsvp: Handles user RSVP to an event.
  • GET /badges: Displays earned badges and achievements for a user.
